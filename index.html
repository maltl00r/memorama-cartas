Introducción

El proyecto consiste en investigar cuál es el lenguaje o los lenguajes de programación más adecuados y comúnmente utilizados para desarrollar un juego de memoria con cartas volteadas. Esta investigación permitirá comprender las opciones más utilizadas en el ámbito del desarrollo de este tipo de juegos, identificando las características técnicas que los hacen apropiados, el entorno en el que funcionan y los recursos que ofrecen.

Justificación

Es importante seleccionar un lenguaje de programación que permita crear un juego de memoria interactivo, eficiente y compatible con el entorno previsto. Para ello, se analizarán sus características técnicas, su ámbito de aplicación y el soporte que ofrece su comunidad de desarrolladores.

Objetivo

Entender qué lenguaje de programación facilita la creación de un juego interactivo, dinámico y fácil de mantener, asegurando que funcione correctamente en los entornos previstos, mediante el análisis de sus características, ámbito de aplicación, comunidad y herramientas disponibles.

JUEGO DE MEMORIA CON CARTAS VOLTEADAS ( TIPO “MEMORAMA CON CARTAS”)

Lenguaje de programación

JavaScript es ideal para este tipo de juego online

Características del lenguaje

  • Imperativo y Estructurado: Javascript es un lenguaje imperativo en el que se van ejecutando las sentencias de manera secuencial. Además Javascript permite una programación estructurada.
  • Tipado débil: En Javascript no se define el tipo de una variable a la hora de instanciarla. El tipo de la variable se asigna atendiendo al valor que le asignemos a la variable. Además si vamos cambiando el valor asignado a la variable, esta podrá cambiar de tipo de datos. Es por esto que al tipado de Javascript, además de débil se le considera como tipado dinámico.
  • Sencillo y muy utilizado: Es un lenguaje muy sencillo, no hace falta tener unos amplios conocimientos de programación para poder hacer un programa en Javascript.
  • Multiplataforma: Al ser un lenguaje interpretado y que se ejecuta tanto en cliente, como en servidor, le permite a Javascript tener múltiples intérpretes en diferentes navegadores (Google Chorme, Microsoft Edge, Safari, Opera,…) que se ejecutan en diferentes sistemas operativos (Windows, Mac, Linux,…).
  • Sintaxis sencilla: Es fácil de entender y escribir, por lo que se pueden crear funciones que muestren o comparen cartas sin complicaciones.
  • Interactividad en tiempo real: Permite que el juego responda al instante cuando el jugador hace clic en una carta.
  • Manipulación del DOM: Facilita cambiar elementos de la página, como voltear cartas o mostrar un mensaje cuando se encuentra un par.
  • Control de eventos: Maneja clics, temporizadores y animaciones, cosas esenciales para un juego de memoria.

Ámbito de aplicación de JavaScript

  • Desarrollo web frontend: Es el lenguaje estándar de los navegadores para crear páginas interactivas, dinámicas y con efectos visuales.
  • Desarrollo web backend: Gracias a entornos como Node.js, se usa para programar servidores y manejar bases de datos.
  • Aplicaciones móviles: Con frameworks como React Native, Ionic o NativeScript se pueden crear apps para Android y iOS.
  • Aplicaciones de escritorio: Usando herramientas como Electrón o Tauri, se desarrollan programas para Windows, macOS y Linux.
  • Videojuegos: Se utiliza en motores como Phaser o Three.js para juegos 2D/3D en web.
  • Internet de las cosas (IoT): Compatible con dispositivos inteligentes y microcontroladores usando plataformas como Johnny-Five.
  • Entornos científicos y de datos: Aunque no es el más común, se usa con librerías como TensorFlow.js para IA y análisis de datos.

Comunidad y herramientas

Se identificó que JavaScript posee una de las comunidades más grandes y activas a nivel mundial. Esto facilita el acceso a material de aprendizaje, ya que existen miles de tutoriales, documentación oficial, ejemplos prácticos y cursos gratuitos o de pago disponibles en línea. Además, foros como Stack Overflow, repositorios en GitHub y comunidades en redes sociales ofrecen un soporte constante, resolviendo dudas y compartiendo soluciones a problemas comunes. También se realizan eventos y conferencias internacionales, como JSConf o NodeConf, que fomentan la innovación y el intercambio de conocimientos.

En lo que respecta a herramientas, el ecosistema de JavaScript es muy amplio y variado. Librerías como Anime.js y GSAP permiten crear animaciones fluidas y transiciones atractivas; frameworks de diseño como Bootstrap o Tailwind CSS simplifican la creación de interfaces modernas y adaptables; mientras que el uso de Vanilla JavaScript (sin frameworks) es suficiente para desarrollar juegos simples, como el clásico juego de memoria con cartas volteadas. Para proyectos más avanzados, existen motores especializados como Phaser.js y librerías de gráficos como PixiJS, que mejoran el rendimiento y permiten incorporar efectos visuales complejos.

Este conjunto de características aporta ventajas significativas al desarrollo del proyecto, entre ellas:

  • Desarrollo ágil: la implementación de la lógica y el diseño del juego puede realizarse en poco tiempo.
  • Pruebas y depuración inmediatas: el código se ejecuta directamente en el navegador, facilitando la identificación de errores.
  • Multiplataforma: el mismo código funciona en computadoras, tabletas y teléfonos móviles sin adaptaciones extensas.
  • Compatibilidad universal: cualquier navegador moderno puede ejecutar el juego sin instalar software adicional.
  • Escalabilidad: la amplia cantidad de librerías y frameworks permite mejorar el proyecto con nuevas funciones de forma sencilla.

En conclusión, la combinación de una comunidad activa, herramientas potentes y facilidad de despliegue convierte a JavaScript en una opción altamente recomendable para el desarrollo de un juego de memoria en entorno web o móviles.

Justificación, alternativas y comparación

Justificación de JavaScript

  • Por lo general no se requiere de una instalación, el juego se ejecuta en el navegador directamente.
  • Es compatible con cualquier dispositivo como una PC, Tablet o celular.
  • Ofrece interactividad inmediata.
  • Permite las animaciones y transiciones muy fluidas.

Lenguaje alternativo: Python con Pygame

Python es un lenguaje de alto nivel de programación, también conocido por su sintaxis clara y legible, además, logra destacar que en su biblioteca encontramos Pygame que nos ayuda a desarrollar juegos en 2D, manejar gráficos, sonidos y eventos de forma más fácil. Esta herramienta es utilizada en la educación y algunos prototipos de juegos para escritorio.

Comparación y escenarios de preferencia

Aspecto JavaScript Python con Pygame
Ejecución Directa en cualquier navegador. Requiere instalación en el dispositivo.
Compatibilidad Web, móvil y escritorio vía navegador. Solo escritorio de forma nativa.
Facilidad de compartir Basta un enlace. Se debe enviar instalador o carpeta del juego.
Curva de aprendizaje Media-baja (HTML, CSS, JS). Baja (Python es muy legible).
Cuándo elegirlo Juego online accesible para todos. Juego local o educativo sin depender de internet.

¿Por qué uno podría ser preferible sobre el otro en un escenario particular?

JavaScript tiende a ser mejor cuando un juego es de fácil acceso para cualquier persona, sin necesidad de instalar nada y que sea compatible para computadoras o dispositivos móviles.

Python con Pygame si el proyecto creado está fomentado al ámbito educativo, local o de escritorio donde instalarlo es relativamente sencillo y es requerido practicar lógica y el manejo de eventos en un entorno controlado.

CONCLUSIÓN

El juego que hemos realizado de las cartas volteadas es práctico y sencillo ya que lo podemos aplicar en la programación para tener experiencias interactivas y divertidas. A través de JavaScript, el juego funciona directamente en el navegador, es accesible para cualquier usuario y además es compatible con múltiples dispositivos. El lenguaje nos ayuda a implementar animación, manejo de eventos y actualizaciones en la interfaz en tiempo real, mejorando constantemente el juego.
